Projects

Jimothy: Autonomous Maze-Navigating Rover

Jimothy is an autonomous rover I developed with three team members using a PiCar kit. Our goal was to program Jimothy to autonomously navigate a maze and locate a target object of a specific color. This project, part of my Computer Elements II course, provided hands-on experience with a range of technologies and skills.

Comps-Hi Chat App

The Comps-Hi Chat App is a Peer-to-Peer network application that enables seamless messaging between peers. It features dynamic network participation, flexible roles, and robust error handling, all operating within a local network using an index server for peer discovery. The project includes both server and client components, with straightforward setup and usage instructions. For a detailed overview, check out the project and blog on my Github.

Foci - Focus App

Foci is a productivity app designed to help users block access to distracting applications and notifications for a specified period, fostering better focus and concentration. Users can add friends, engage in friendly competition, and motivate each other to stay on task. As users complete more focus sessions, they earn medals to showcase their achievements on their profiles.

JimothyV2: Honing POV Display

JimothyV2 is a continuation to my groups previous autonomous project. We made some modifications to Jimothy’s original design to be able to add a POV display to the top of our robot. The POV display is made of 9 LEDs and an arduino that are spun by a DC motor in order to display different images and text.
